# mlops-credit-scoring
# MLOps Credit Scoring Project

## Описание проекта

Данный проект посвящён промышленному развертыванию модели кредитного скоринга с использованием практик MLOps.  
Цель работы — продемонстрировать полный жизненный цикл модели машинного обучения: от подготовки модели к продакшену до автоматизации доставки и базовой инфраструктурной организации.

Проект выполнен в рамках дисциплины  
**«Автоматизация процессов доставки и развертывания моделей машинного обучения»**.

## Этап 1. Подготовка модели к промышленной эксплуатации

Нейронная сеть
•	Реализована простая нейросеть на PyTorch
•	Модель обучена и сохранена в формате .pt

Конвертация в ONNX
•	Модель успешно конвертирована в формат ONNX
•	Проведена проверка корректности конвертации
•	Максимальное абсолютное отклонение между PyTorch и ONNX: ~1e-8

Оптимизация модели
•	Применена динамическая quantization
•	Выполнено сравнение размера модели и latency до/после оптимизации

Вывод:

ONNX-модель корректна и готова к использованию в продакшене.
Оптимизация показала компромисс между скоростью и размером модели.


## Этап 2. Infrastructure as Code (Terraform)
•	Создана модульная структура Terraform
•	Выделены отдельные модули:
•	VPC
•	Kubernetes
•	Storage
•	Monitoring
•	Конфигурация демонстрирует подход IaC для облачного развёртывания

Фактическое применение Terraform в облаке не выполнялось, так как целью этапа является демонстрация архитектуры.

## Этап 3. Контейнеризация и оркестрация
•	Создан Dockerfile для приложения
•	Подготовлены Kubernetes-манифесты:
•	Deployment с rolling update
•	Service для доступа к приложению
•	ConfigMap для конфигурации

## Этап 4. CI/CD Pipeline
	•	Настроен GitHub Actions pipeline
	•	Pipeline включает этапы:
	•	checkout кода
	•	установка зависимостей
	•	запуск проверки модели (inference.py)
	•	Pipeline автоматически запускается при push в репозиторий

CI/CD используется для автоматической проверки корректности модели и кода.
